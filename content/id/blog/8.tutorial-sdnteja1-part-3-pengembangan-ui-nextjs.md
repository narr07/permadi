Setelah menyiapkan pondasi data di Sanity CMS, sekarang saatnya kita membangun "wajah" dari website **SDN Teja 1**. Kita akan menggunakan gaya desain **Neo-brutalist** yang memberikan kesan modern, berani, dan unik.

## Mengenal Gaya Neo-brutalist

Neo-brutalisme dalam desain web ditandai dengan penggunaan:
- **Warna yang kontras:** Penggunaan warna latar belakang yang terang dengan elemen yang mencolok.
- **Border tebal:** Garis tepi hitam yang tebal (`border-4` atau `border-foreground`).
- **Shadow yang solid:** Bayangan yang tidak lembut (blur), melainkan solid dan tegas.
- **Tipografi yang berani:** Font yang besar dan *heavy* untuk menarik perhatian.

## Membangun Komponen Navbar

Navbar adalah elemen navigasi utama. Kita ingin navbar yang responsif dan tetap mengikuti gaya neo-brutalist.

Contoh potongan kode `components/Navbar.tsx`:

```tsx
export default function Navbar() {
  return (
    <nav className="border-b-4 border-foreground bg-background sticky top-0 z-50">
      <div className="max-w-7xl mx-auto px-4 flex justify-between items-center h-20">
        <Link href="/" className="text-2xl font-black tracking-tighter">
          SDN TEJA 1
        </Link>
        <div className="hidden md:flex gap-8 items-center">
          <Link href="/" className="font-bold border-2 px-3 py-2 hover:border-b-4 hover:border-main">
            Home
          </Link>
          {/* Link lainnya... */}
        </div>
      </div>
    </nav>
  )
}
```

Perhatikan penggunaan `border-b-4 border-foreground` yang memberikan garis bawah tebal yang menjadi ciri khas utama desain kita.

## Komponen Hero dengan Grid Background

Untuk bagian **Hero**, kita ingin sesuatu yang mencolok. Kita bisa menggunakan pola *grid* di latar belakang untuk menambah tekstur retro.

```tsx
export function Hero() {
  return (
    <section className="w-full py-24 bg-[linear-gradient(to_right,#0000001a_1px,transparent_1px),linear-gradient(to_bottom,#0000001a_1px,transparent_1px)] bg-size-[32px_32px]">
      <div className="container mx-auto text-center">
        <h1 className="text-5xl md:text-7xl font-black tracking-tighter drop-shadow-md">
          Membangun Masa Depan di SDN Teja 1
        </h1>
        <p className="mt-4 text-xl font-medium">
          Sekolah berprestasi dengan fasilitas modern dan lingkungan belajar yang menyenangkan.
        </p>
        <div className="mt-8 flex justify-center gap-4">
          <Button size="lg">Daftar Sekarang</Button>
          <Button variant="secondary" size="lg">Lihat Profil</Button>
        </div>
      </div>
    </section>
  )
}
```

## Tips Tailwind CSS v4

Dalam **Tailwind CSS v4**, banyak optimasi yang dilakukan pada proses build. Pastikan Anda memanfaatkan variabel CSS untuk kustomisasi warna agar tema website konsisten di seluruh komponen.
