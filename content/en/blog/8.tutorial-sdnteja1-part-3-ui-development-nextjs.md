After setting up the data foundation in Sanity CMS, it's now time to build the "face" of the **SDN Teja 1** website. We will use the **Neo-brutalist** design style, which provides a modern, bold, and unique impression.

## Getting to Know Neo-brutalist Style

Neo-brutalism in web design is characterized by the use of:
- **Contrast colors:** Using bright background colors with striking elements.
- **Thick borders:** Bold black edges (`border-4` or `border-foreground`).
- **Solid shadows:** Shadows that are not soft (blurred), but solid and sharp.
- **Bold typography:** Large and heavy fonts to draw attention.

## Building the Navbar Component

The Navbar is the main navigation element. We want a navbar that is responsive and still follows the neo-brutalist style.

Example code snippet from `components/Navbar.tsx`:

```tsx
export default function Navbar() {
  return (
    <nav className="border-b-4 border-foreground bg-background sticky top-0 z-50">
      <div className="max-w-7xl mx-auto px-4 flex justify-between items-center h-20">
        <Link href="/" className="text-2xl font-black tracking-tighter">
          SDN TEJA 1
        </Link>
        <div className="hidden md:flex gap-8 items-center">
          <Link href="/" className="font-bold border-2 px-3 py-2 hover:border-b-4 hover:border-main">
            Home
          </Link>
          {/* Other links... */}
        </div>
      </div>
    </nav>
  )
}
```

Notice the use of `border-b-4 border-foreground` which provides the thick underline that is the main hallmark of our design.

## Hero Component with Grid Background

For the **Hero** section, we want something eye-catching. We can use a grid pattern in the background to add a retro texture.


## Tailwind CSS v4 Tips

In **Tailwind CSS v4**, many optimizations have been made to the build process. Make sure you utilize CSS variables for color customization so that the website theme is consistent across all components.
